# Demo e Templates — Aegis Engine

## Demo completo

Pasta: `examples/demo-platformer/`

Inclui 3 fases pequenas de plataforma com player, inimigos, tilemap, colisão automática, pathfinding, coletáveis, HUD, shaders, partículas, áudio e gamepad.

Rodar:

```bash
aegis run examples/demo-platformer
```

## Templates

Pastas:

- `templates/platformer`
- `templates/topdown`
- `templates/puzzle`

CLI planejada/implementada no código:

```bash
aegis new platformer meu-jogo
aegis new topdown meu-rpg
aegis new puzzle meu-puzzle
```

## Nota honesta

O ambiente desta geração não tinha `dotnet`, então não foi possível compilar executáveis Windows/Linux nem gravar GIF real de gameplay. O código e a estrutura foram entregues para validação local.
